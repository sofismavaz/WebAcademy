Listas

As listas são elementos HTML usados para agrupar itens relacionados. Existem dois tipos principais de listas:

  • Lista não ordenada (<ul>): usada para agrupar itens sem uma ordem específica.
  • Lista ordenada (<ol>): usada para agrupar itens em uma ordem específica.

Exemplo de Lista Não Ordenada

  • <div>: Um contêiner genérico para agrupar elementos.
  • <span>: Um contêiner em linha para agrupar texto ou outros elementos em linha.
